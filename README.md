#  **News Site with Next.js**

Welcome to the **News Site** project! A modern news platform built with **Next.js**, designed to deliver timely news updates from various categories. The site allows users to explore the latest articles, dive into yearly archives, and seamlessly navigate through different news sections. Built with **Next.js**, it ensures an optimal experience through server-side rendering (SSR) and static site generation (SSG).

---

##  **Key Features**

###  **Latest News Section**
- Stay up-to-date with breaking news and recent events, organized in chronological order.

###  **Yearly Archives**
- Access historical news from specific years (e.g., 2023, 2022) and dive into past articles.

###  **All News Section**
- Explore a vast array of news articles categorized by topics like **World**, **Sports**, **Technology**, **Health**, and more.

---

##  **Key Functionalities**

###  **Group Routing**
- Dynamically handles various content categories, making it easy for users to navigate through sections like **World**, **Sports**, **Technology**, etc.

###  **Parallel Routes**
- Allows displaying multiple categories side-by-side, giving users the flexibility to view different news sections at once.

###  **Nested Routing**
- Utilizes hierarchical structure for articles, providing an intuitive user journey for detailed content exploration.

---

##  **Navigation Structure**

The **News Site** features an intuitive and user-friendly navigation system:

- **Main Navigation Bar**: Links to **Home**, **Categories**, **News**, and **Archive** pages.
---

##  **Technologies Used**

- **Next.js**: A React framework that powers SSR and SSG for a fast, SEO-optimized experience.
- **CSS**: For styling and ensuring the website is visually appealing and responsive across all devices.
- **JavaScript (JS)**: Used for handling dynamic interactions, routing, and client-side functionality.
- **Vercel**: The platform for easy deployment and hosting of Next.js applications, offering automatic scaling and serverless functions.
